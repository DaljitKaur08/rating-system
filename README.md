# Rating System

Assignment 2 - Components and Props

This project is a React application with a star rating system and a dialog popup.

## Features

- Five clickable stars
- Rating text changes when a star is selected
- Dialog popup opens and closes
- Uses React components and props
- Uses the useState hook

## Components

### Star
Displays a star icon.

### StarRating
Displays five stars and rating text.

### Dialog
Displays a popup dialog box.

## Technologies

- React
- JavaScript
- CSS
- React Icons
